## Naming Convention

Naming Convention for Machine User Presets

`\<PRINTER> \<NOZLESIZE> - \<CHANGES SUMMARY>`

e.g

`Bambu Lab X1 Carbon 0.4 nozzle - REDUCED CURRENT`
